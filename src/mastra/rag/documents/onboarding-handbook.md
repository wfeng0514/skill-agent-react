## 1. 公司介绍

### 1.1 公司简介

我们是一家专注于 AI 基础设施和智能应用开发的科技公司，总部位于深圳南山区科技园。公司成立于 2020 年，目前团队规模约 150 人，产品覆盖 AI Agent 平台、低代码平台和智能办公三大板块。

## 2. 入职注意事项

### 2.1 入职第一天必办事项

1. **领取办公设备**：到行政部领取笔记本电脑（MacBook Pro M3）、显示器、机械键盘
2. **开通系统权限**：IT 部门会为你开通以下账号：
   - 企业邮箱（@company.com）
   - GitLab 代码仓库（gitlab.company.com）
   - 飞书（即时通讯和 OA）
   - 内部 Wiki（wiki.company.com）
3. **完成入职培训**：在线学习平台完成必修课程（入职首周内完成）

### 2.2 入职导师（Buddy）

每位新员工都会分配一位入职导师，通常是你所在团队的高级工程师。导师的职责：

- 帮助你熟悉代码仓库和开发流程
- 解答技术问题，带你熟悉技术栈
- 每周进行一次 1v1 沟通
- 入职满月时协助完成试用期第一次评估

### 2.3 试用期说明

**试用期时长：**

| 职级         | 试用期时长 |
| ------------ | ---------- |
| 初级工程师   | 3 个月     |
| 中高级工程师 | 2 个月     |
| 架构师及以上 | 1 个月     |

**试用期目标：**

- **第一个月**：熟悉代码仓库和开发流程，独立完成 2-3 个小需求
- **第二个月**：独立负责一个完整功能模块，参与 Code Review
- **第三个月（如适用）**：主导一个小型项目，展示技术能力和协作能力

**试用期评估维度：** 技术能力、协作能力、学习态度、文化契合度

## 3. 考勤与假期

### 3.1 作息时间

- **弹性工作制**：核心工作时间为 10:00-12:00 和 14:00-18:00
- 建议到岗时间：9:30-10:00
- 午休时间：12:00-14:00
- **每周五 16:00-17:00**：技术分享时间（非强制，鼓励参加）

### 3.2 远程办公

- 每月有 **4 个远程办公日**，无需审批
- 远程办公需保持飞书在线，及时响应消息
- 重要会议建议到公司参加
- 远程办公超过连续 3 天需要主管审批

### 3.3 假期类型

| 假期类型 | 天数     | 说明                               |
| -------- | -------- | ---------------------------------- |
| 年假     | 15 天/年 | 入职即享，按比例折算首年天数       |
| 病假     | 不限     | 3 天内无需证明，3 天以上需医院证明 |
| 事假     | 按需     | 需提前 1 天在飞书 OA 申请          |
| 调休     | 按需     | 加班产生的调休额度，有效期 6 个月  |
| 产假     | 178 天   | 按法规执行                         |
| 陪产假   | 15 天    | 按法规执行                         |
| 育儿假   | 5 天     | 子女 3 周岁内额外享有              |

### 3.4 请假流程

飞书 → OA 审批 → 选择假期类型 → 填写起止日期 → 提交审批

## 4. 规章制度

### 4.1 信息安全红线

以下行为严格禁止，违反可能导致解除劳动合同：

1. 将生产环境数据库密码、API 密钥等敏感信息提交到公开代码仓库
2. 将公司源代码上传到个人 GitHub 账号或任何外部平台
3. 未经授权访问或导出用户数据
4. 在公共场合讨论公司未公开的技术方案或业务数据
5. 使用未经审批的第三方 SaaS 服务处理公司数据

### 4.2 代码规范

- **TypeScript 项目**：遵循 ESLint + Prettier 配置
- **Git 提交规范**：使用 Conventional Commits 格式

```
 feat: 新增用户注册功能
 fix: 修复登录页面样式错位问题
 docs: 更新 API 文档
 refactor: 重构数据库连接池模块
 test: 补充用户模块单元测试
 chore: 升级依赖版本
```

- **Code Review**：每个 MR 至少需要 1 位同事 Review 通过后才能合并

## 6. 离职交接

### 6.1 离职申请

- **试用期内**：提前 3 天提交离职申请
- **正式员工**：提前 30 天提交离职申请
- **申请方式**：飞书 OA → 离职申请 → 填写离职原因和最后工作日

### 6.2 离职交接清单

| 交接项         | 对接人      | 完成标志                  |
| -------------- | ----------- | ------------------------- |
| 代码与文档交接 | 直属主管    | 代码合并、文档更新        |
| 设备归还       | IT 支持     | 归还电脑、显示器等设备    |
| 资产清算       | 财务        | 还清借款、报销完成        |
| 权限回收       | IT 支持     | 确认所有系统权限已注销    |
| 知识传递       | 主管/接任者 | 完成至少 1 次知识传递会议 |
| 门禁卡归还     | 行政        | 归还门禁卡、钥匙          |

### 6.3 离职后事项

- **薪资结算**：离职后 5 个工作日内发放
- **离职证明**：离职当天可领取
- **社保公积金**：每月 15 日前离职的，当月停止缴纳；15 日后离职的，次月停止
- **期权处理**：按期权协议约定处理

### 7.5 常见问题

**Q：公司用什么技术栈？**

A：后端以 Node.js/TypeScript 为主，前端 React + TypeScript，AI 平台使用 Python。数据库主要用 PostgreSQL 和 Redis。基础设施基于 Kubernetes。

**Q：加班多吗？**

A：我们推崇高效工作，不鼓励无效加班。项目冲刺期可能会有适度加班，加班可申请调休或按加班费标准发放。

**Q：如何申请调薪？**

A：每年有两次调薪窗口（3 月和 9 月），主管会根据绩效评估结果提出调薪建议。

**Q：电脑坏了怎么办？**

A：飞书联系 @it-support，办公时间内通常 30 分钟内响应。

**Q：公司有没有零食？**

A：有的！茶水间有免费咖啡、茶、零食，每周三下午还有下午茶。
